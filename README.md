# 웹툰발굴단 - 베스트 도전작의 승격여부를 데이터로 알아보자 🔍

### ✅ 개요

2022년 12월 말에 수집한 데이터를 기반으로 베스트도전 웹툰의 승격여부를 예측해보는 ML프로젝트
<br>

<ol>
<li>
수집대상 데이터 <br>
베스트도전에서 5회 이상 회차가 존재하는 웹툰의 데이터(2022년도 12월 15일 기준 확인되는 모든 장르의 웹툰)
